## Problem 1

``` r
hw_df = tibble(
  vec_rnorm = rnorm(10),
  vec_logical = vec_rnorm > 0,
  vec_char = c("A","B","C","D","E","F","G","H","I","J"),
  vec_factor = factor(c("small", "small","large","medium","large","small","medium","small","large","large"))
)
```
